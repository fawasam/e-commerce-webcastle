## Table of contents


1. Clone the repository:
  ```bash https://github.com/fawasam/e-commerce-webcastle.git
   ```
3. Navigate to the project directory:
   ```bash
   cd backend
   cd frontend
   ```
4. Install dependecy
   ```bash
   npm install
    
   ```
 5. Run the following command to start the application:
  ```bash
    backend
       npm run dev
    frontend
        npm run prod
   ```
